-- SUMMARY --

This is a module that implements Views display plugin to output the result of a view into PHP Array. The PHP Array will later can be used by other modules for further processing.

-- REQUIREMENTS --

Views 6.x-2.x or newer.

-- INSTALLATION --

Install as per usual module.

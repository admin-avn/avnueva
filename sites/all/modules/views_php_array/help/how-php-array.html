Adding PHP Array Display

To add PHP Array Display, add it as per normal (as what you would do to add Page, Block, Feed display) in the Views UI. The PHP Array Style plugin will be selected automatically whenever PHP Array Display is added.

Previewing PHP Array Display

To preview the result, select PHP Array in the live preview option and continue with the previewing. Devel module is required to enable previewing of the array in Krumo format.
